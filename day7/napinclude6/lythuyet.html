Sự khác biệt của include, include_once, require,require_once

    Sự giống nhau: 4 hàm này đều dùng để nạp 1 file vào file khác
    Sự khác nhau:
    -include( Khi mà quá trình nạp file bị lỗi thì chương trình vẫn chạy tiếp)
    -require:( Khi mà quá trình nạp file bị lỗi thì chương trình sẽ dừng chạy)
    -include_once( Giống include nhưng chỉ nạp file 1 lần ngay cả khi bị gọi nhiều lần)
    -require( giống Require nhưng chỉ nạp file 1 lần ngay cả khi gội nhiều lệnh)
